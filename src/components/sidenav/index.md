# Sidenav 侧导航 

对 Nav 的一个封装，用于页面侧边栏。

* `<Sidenav>` 侧导航组件。
* `<Sidenav.Header>` 导航顶部内容区域。
* `<Sidenav.Body>` 导航选项区域。

## 获取组件

```js
import { Sidenav } from 'rsuite';
```

## Props

### `<Sidenav>`

| 属性名称       | 类型                                 | 默认值    | 描述                 |
| -------------- | ------------------------------------ | --------- | -------------------- |
| appearance     | enum: 'default', 'inverse', 'subtle' | 'default' | 菜单外观             |
| classPrefix    | string                               | 'sidenav' | className 前缀       |
| componentClass | React.ElementType                    | 'div'     | 为组件自定义元素类型 |
