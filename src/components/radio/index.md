## API
### `<Radio>`
| 属性名称     | 类型          | 默认值   | 描述  |
|----------|-------------|-------|-----|
| id       | React.ElementType |       |     |
| name     | string      |       |     |
| inline   | boolean     | false |     |
| title    | string      |       |     |
| disabled | boolean     | false |     |
| checked  | boolean     |       |     |
| onClick  | function    |       |     |
| onChange | function    |       |     |

### `<RadioGroup>`

| 性名称    | 类型      | 默认值   | 描述  |
|--------|---------|-------|-----|
| name   | string  |       |     |
| inline | boolean | false |     |
