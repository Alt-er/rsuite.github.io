```js

```

## Props

### `<Divider>`

| Property    | Type`(default)`      | Description                   |
| ----------- | -------------------- | ----------------------------- |
| classPrefix | string `('divider')` | prefix of className           |
| vertical    | boolean              | vertical dividing line        |
| children    | React.Node           | set text in the dividing line |
