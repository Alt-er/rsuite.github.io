## Props

### `<Divider>`

| 属性名称    | 类型`(默认值)`       | 描述               |
| ----------- | -------------------- | ------------------ |
| classPrefix | string `('divider')` | className 前缀     |
| vertical    | boolean              | 垂直分割线         |
| children    | React.Node           | 设置分割线上的文本 |
